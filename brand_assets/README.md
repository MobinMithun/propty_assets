# Brand Assets

Place all brand files here before designing. Claude checks this folder first.

- **Logos** — SVG preferred, PNG acceptable (2x resolution minimum)
- **Color palette** — a `colors.md` or `colors.json` with exact hex values
- **Typography guide** — font names, weights, size scales
- **Style guide** — any reference screenshots, mood boards, or design tokens
- **Images** — hero photos, property images, team photos
